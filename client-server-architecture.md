What is Express ? 

Ans : 

- Lightweight and fast Node.js web framework.
- Simplifies HTTP server and API development.

------------------------------------------------------------

 Why Use Express.js?

Ans :

-Minimalistic and flexible
-Huge community and ecosystem
-Compatible with many third-party libraries

REST APIs and Their Components :

-REST = Representational State Transfer
-Architecture style using HTTP to access and manipulate data
-Stateless and resource-based
-------------------------------
-->RESTful Principles :

-Statelessness: Each request is independent.
-Client-server: UI and backend are separated.
-Cacheable: Responses can be cached.
-Uniform Interface: Standard HTTP methods.
--------------------------------
Components  	 |        Description
-----------             ---------------
Endpoints	        Resource URLs (e.g., /api/users)
Methods	            GET, POST, PUT, DELETE
Request Headers	    Metadata (e.g., Content-Type, Authorization)
Request Body	    Data sent to the server (JSON format)
Response Body	    JSON data sent back to the client
Status Codes	    HTTP codes like 200, 201, 404, 500
