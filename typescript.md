## What is TypeScript?

TypeScript is a language developed and maintained by Microsoft. It offers:

- Static type checking
- Support for modern JavaScript features
- Rich IDE support
- Optional typing

TypeScript files use the `.ts` extension and compile into regular JavaScript using the TypeScript compiler (`tsc`).
